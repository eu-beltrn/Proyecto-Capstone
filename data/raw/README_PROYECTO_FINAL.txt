# Dataset Masivo Compatible - Proyecto Final de Ingeniería de Datos

Este paquete mantiene exactamente la misma estructura que los archivos de muestra iniciales.

Archivos incluidos:
1. ventas_masivo.csv
2. productos_masivo.xlsx
3. clientes_masivo.json
4. inventario_masivo.db
5. api_marketing_response_masivo.json

IMPORTANTE:
La estructura de columnas y tablas se mantiene igual que en las muestras iniciales.
Los errores están únicamente en los valores, no en los nombres de columnas ni en la estructura.

El objetivo del estudiante es:
- cargar datos masivos,
- perfilar la calidad de datos,
- limpiar errores,
- transformar información,
- validar reglas de negocio,
- integrar las fuentes,
- crear el Data Warehouse,
- generar consultas SQL, KPIs y visualizaciones.

No se deben modificar manualmente los archivos fuente.
Toda limpieza debe hacerse con Python, SQL y procesos automatizados.
